                <h1 class="class">Caricature::Isolation</h1>

                <div id="description">
                        <p>

Instead of using confusing terms like Mocking and Stubbing which is basically the same. Caricature tries to unify those concepts by using a term of Isolation. When you&8217;re testing you typically want to be in control of what you&8217;re testing. To do that you isolate dependencies and possibly define return values for methods on them. At a later stage you might be interested in which method was called, maybe even with which parameters or you might be interested in the amount of times it has been called.
