                                <div class="method-description">

                                        <p>




Finds an expectation in the collection. It matches by method_name
first. Then it tries to figure out if you care about the arguments. You can
say you don&8217;t care by providing the symbol :any as first
argument to this method. When you don&8217;t care the first match is
being returned When you specify arguments other than :any it will
also try to match the specified arguments in addition to the method name.
It will then also return the first result it can find.
